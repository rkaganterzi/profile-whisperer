## 1. Executive Summary

**Profile Whisperer**, Instagram profillerindeki fotoğrafları yapay zeka ile analiz ederek karakter çıkarımı yapan ve kullanıcıya kişiselleştirilmiş sohbet başlatıcıları öneren bir mobil uygulamadır.

| | |
|---|---|
| **Hedef** | Z kuşağının flört ve sosyalleşme deneyimini eğlenceli hale getirmek |
| **Platform** | iOS & Android (Flutter) |
| **Gelir Modeli** | Freemium + Mikro ödemeler |
| **Hedef Kitle** | 18-28 yaş, sosyal medya aktif kullanıcıları |
| **Pazar** | Global (English-first) + Türkçe |
| **Geliştirme** | Solo developer + Claude Code |
| **Bütçe** | MVP: ₺0 (Free tier stack) |

### Geliştirme Ortamı: Claude Code Bridge (API Maliyetsiz)

Geliştirme aşamasında Claude API maliyetinden kaçınmak için "köprü" mimarisi:

```
┌──────────────────────────────────────────────────────────────────┐
│                    DEVELOPMENT MODE                               │
├──────────────────────────────────────────────────────────────────┤
│                                                                   │
│  ┌─────────┐    ┌─────────┐    ┌─────────────┐    ┌───────────┐ │
│  │ Flutter │───▶│ FastAPI │───▶│  ./requests │───▶│  Claude   │ │
│  │   App   │    │ Bridge  │    │  ├─ image   │    │   Code    │ │
│  └─────────┘    └─────────┘    │  └─ prompt  │    │   (CLI)   │ │
│       ▲              ▲         └─────────────┘    └─────┬─────┘ │
│       │              │                                   │       │
│       │              │         ┌─────────────┐           │       │
│       │              └─────────│ ./responses │◀──────────┘       │
│       │                        │  └─ result  │                   │
│       │                        └──────┬──────┘                   │
│       │                               │                          │
│       └───────────────────────────────┘                          │
│                                                                   │
└──────────────────────────────────────────────────────────────────┘
```

**Akış:**
1. Flutter fotoğrafı FastAPI'ye gönderir
2. FastAPI `./requests/{id}/` klasörüne `image.jpg` + `prompt.txt` yazar
3. Claude Code (manuel) klasörü okur, analiz yapar
4. Claude Code `./responses/{id}/result.json` yazar
5. FastAPI polling ile response'u bekler, Flutter'a döner

**Avantajları:**
- Geliştirme maliyeti: ₺0
- Prompt'lar gerçek ortamda test edilir
- Production'a geçiş: Sadece AI service değişir

**Production Geçişi:**
```python
# Development
ai_service = ClaudeCodeBridge()

# Production
ai_service = ClaudeAPI(api_key=CLAUDE_API_KEY)
```

### API Endpoint Yapısı

```
POST /api/v1/analyze

GET  /api/v1/analyze/{id}

GET  /api/v1/user/history
  - Kullanıcının analiz geçmişi
```
